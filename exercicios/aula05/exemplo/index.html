HTML é uma linguagem de programação para construção de páginas web.

Com HTML, é possível definir o conteúdo e a estruturação do documento

Tags em HTML permitem a criação de links, imagens e tabelas.

JavaScript é uma linguagem de programação usada
para criar interatividade na web.

CSS é uma linguagem usada para estilizar os documentos HTML.

Com CSS, é possível comtrolar a aparência e o layout da página.
